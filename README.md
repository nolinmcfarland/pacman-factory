## Installation
 
Here's how to get started with Pac-Man Factory:
 
1. Clone the project!
1. Run `index.html`.
 
## Usage

- Click "Add Pac-Man" to add a new Pac-Man to the screen!
- Click "Reset" to remove all Pac-Men.
